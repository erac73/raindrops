What is Rain Drops?

Rain Drops is a distributed information storage model based on threshold cryptography (Shamir's Secret Sharing). Data is fragmented into cryptographic micro-units — called drops — that individually carry no meaning. The original data can only be reconstituted when a sufficient number of drops converge under verified conditions.

This is not encryption layered on top of storage. Confidentiality is a structural property of the model.

🔐

Perfect Secrecy

With fewer than K drops, the secret is information-theoretically hidden. Not even quantum computers can break it.

🧩

K-of-N Threshold

Data is split into N drops. Any K drops can reconstruct; K-1 reveal nothing. Resilient to node failures.

⏱️

Temporal Expiry

Each drop has a TTL. Expired drops are automatically deleted. Time-bound access control by design.

✅

Integrity Verified

Each drop carries an HMAC-SHA256 MAC. The Witness Node verifies all drops before reconstruction.

📦

Multi-Arch Docker

ARM64 and AMD64 images built automatically. Deploy on Raspberry Pi, x86 servers, or cloud.

📊

Prometheus Metrics

All nodes expose metrics, health checks, Swagger docs, and a web dashboard for monitoring.

Components

  • raindrops-core — Cryptographic library (SSS, AES-GCM, HMAC, RainMap)
  • raindrops-storage — Spring Boot storage node with replication and TTL reaper
  • raindrops-witness — Stateless verification node for secure reconstruction
  • RainClient SDK — High-level Java client for DROP/RECONSTRUCT operations

Quick Start

git clone https://github.com/erac73/raindrops-fase1.git
cd raindrops-fase1
mvn -f raindrops-fase1/raindrops/pom.xml install
docker compose -f docker-compose.yml up -d --build

Deployment

Three storage nodes (ports 9081-9083) + one witness node (port 9080) orchestrated via Docker Compose. Native ARM64 support for Raspberry Pi.
